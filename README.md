#  Calculator

A sleek, modern calculator built with HTML, CSS and JavaScript.

##  About The Project
A fully functional calculator with a clean dark-themed design. Supports basic arithmetic operations with a smooth user interface.

##  Features
- Addition, subtraction, multiplication, division
- Percentage calculation
- Clear and delete functions
- Keyboard support
- Responsive dark theme design

##  Built With
- HTML
- CSS
- JavaScript
